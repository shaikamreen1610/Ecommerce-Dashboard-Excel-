##  Project Overview  
This project focuses on analyzing **e-commerce sales data using Microsoft Excel** to extract meaningful business insights. The dashboard presents key performance indicators and trends to support data-driven decision-making.

---

##  Tools & Technologies Used
- Microsoft Excel  
  - Pivot Tables  
  - Pivot Charts  
  - Slicers & Filters  
  - Conditional Formatting  
  - Excel Functions  
  - Data Cleaning Techniques  

---

##  Dataset Description  
The dataset includes transactional e-commerce data with the following columns:

- Invoice Number  
- Stock Code  
- Product Description  
- Quantity  
- Invoice Date  
- Unit Price  
- Customer ID  
- Country  

---

##  Data Cleaning Steps
- Removed duplicate records  
- Handled missing values (especially Customer ID and product descriptions)  
- Corrected data types (dates and numeric columns)  
- Removed invalid or negative quantity values  
- Created calculated columns for analysis  
- Standardized product descriptions  

---

##  Key KPIs & Metrics
- Total Sales  
- Total Revenue  
- Total Orders  
- Unique Customers  
- Average Order Value  
- Top-Selling Products  
- Sales by Country  
- Monthly Sales Trend  
- Revenue Contribution Analysis  

---

##  Dashboard Features
- Interactive slicers for filtering by date, country, and product  
- KPI cards for quick performance overview  
- Dynamic charts that update with filters  
- Clean and user-friendly layout  
- Trend and comparison analysis  

---

##  Insights Generated
- Identified top-performing products and regions  
- Analyzed monthly and yearly sales trends  
- Highlighted high-value customers  
- Detected underperforming products  
- Enabled data-driven business insights
